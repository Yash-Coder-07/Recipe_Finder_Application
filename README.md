# 🥗 Recipe Finder Web App

A modern and interactive recipe search application built using HTML, CSS, and JavaScript. It lets users search meals by name or discover random recipes, with a stylish UI and responsive design. All data is fetched from [TheMealDB API](https://www.themealdb.com/api.php).

---

## 🔗 Live Demo

**👉 [View Live Demo](https://recipe-finder-application-theta.vercel.app/)**  


---

## 🚀 Features

- 🔍 Search recipes by name
- 🎲 Get a random recipe with one click
- 🖼 Stylish Unsplash background with overlay
- 📱 Responsive layout for all devices
- 🍳 View recipe details in a popup modal
  - Ingredients
  - Instructions
  - Category and Area
  - YouTube video tutorial
  - Source link

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **API**: [TheMealDB](https://www.themealdb.com/api.php)
- **Design**: Google Fonts (Poppins), Unsplash Background
